## Getting Started


- npm
- Node.js
- Database working with Sequelize (postgres, mysql, mariadb, sqlite3 or tedious)

### Installation


1. Install NPM packages

   ```sh
   npm install
   ```

2. Create `.env` file

   ```sh
   cp .env.example .env
   ```
